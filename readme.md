# Featured Project

## EcoInsight – Carbon Footprint Tracker

EcoInsight is a frontend web application developed to spread awareness about environmental sustainability by helping users estimate and visualize their carbon footprint based on daily activities.

The project focuses on making environmental data easy to understand through interactive charts, intuitive user interfaces, and meaningful insights that encourage sustainable lifestyle choices.

### Technologies Used

This project was built using **HTML, CSS, JavaScript, React, and Chart.js**, combining modern frontend development practices with interactive data visualization to deliver a responsive and engaging user experience.
